# Village & Pillage

《程序设计基础》2020秋大作业

## Usage

1. First, clone this repository
2. Run `npm install`
3. Run `npm run build`
4. Game is ready in dist folder
5. Deploy it to a server or just run it locally.

How to Run it locally?(one possible way)

```sh
npm install -g light-server
light-server -s ./dist
```
